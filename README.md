# USMA-AY26-2-MA289-Coey
A record of my work from MA289: Intro to Statistical Learning, taken at the US Military Academy in the Spring of 2026. In total, there are notebooks from 3 projects and 2 assignments, with the final project having 2 notebooks just to help with my workflow. The datasets used in some of the notebooks can also be found, along with my write-up for the final project. A brief summary of the projects is below:

Assignment 1: Performing exploratory data analysis on a dataset of bikeshare daily usage over the course of 2 years in Washington DC

Project 1: Using bootstrapping to generate an estimate for the population parameter of differences in acceptance rates between public and private US universities

Project 2: Using logistic regression and KNN to perform classification on the MNIST footwear image dataset

Assignment 2: Creating a synthetic dataset to stress-test ridge regularization, demonstrating how ridge works best when paired with standardization

Project 3: Applying the findings from Assignment 2 to a real housing dataset and tracking how feature selection changes with different amounts of ridge regularization
